The Elis Platform

Mobile services for energy efficiency

This project focuses on exploring the potential of mobile services on mobile devices (e.g. smart phones and tablets) to promote energy efficiency in existing buildings. I led the software development work on a platform that integrates existing technologies from a number of companies including E.On and Schneider-Electric.

The platform is developed in Java using the OSGI framework for module composition. It exposes a HTTP API which 3rd-party developers can use to develop services for energy efficiency. What’s special about the API is that it integrates data from multiple providers and makes it easier to use through useful abstractions.

In the future, hopefully the project will be released as open source, but as of now it is not. The project is run by Malmö University, Lund University, and Blekinge Tekniska Högskola together with several companies in southern Sweden.

Raft in Scala/Akka

An algorithm for solving the consensus problem

This is an implementation of the Raft algorithm. I wanted to explore the implementation intricacies of the algorithm as it claims to be easier to understand than Paxos. Having developed an implementation of Paxos in a university course project I can certainly say Raft is easier to understand and map to a working prototype.

The implementation makes extensive use of Akka’s FSM traits and has a fairly extensive test suite.

Inductive graphs in Scala

An experiment in writing purely functional graphs

This mini-project explored functional data structures and in particular graphs using Scala. The result of the project is an implementation of Inductive Graphs based on Martin Erwig's paper which I contributed to an open-source collection of functional data structure implementations called Scalacaster

MAPI

Transform those HTTP calls

Mapi is a proxy which lets you selectively intercept and re-route HTTP calls. It is a way to modify a part of an existing HTTP api leaving other methods untouched. I developed the relatively short Node.js script as a way to prototype some modifications we were doing with one of the providers API in the Elis project. For example, MAPI allowed us to inject responses from physical devices that were not available through the provider's API.

Thesis Project

Online recommendations at scale with matrix factorisation

As my final project at the European Master in Distributed Computing programme, run joinly by KTH in Stockholm, IST in Lisbon, and UPC in Barcelona, I developed a system to serve video recommendations at large scale.

The system was the first I developed in Scala and makes some use of Akka to coordinate and distribute the computations in a small cluster. Experiments showed that the system could serve up to 600 requests/second with a latency below 30 ms.

The project was carried out at Tuenti, a social network in Spain, together with Toni Cebrian and his colleagues from Telefonica Research.

About Me

I'm a freelance backend developer working mostly in Java, Python, Scala, and Erlang. Occasionally I dabble in other languages and I don't see myself as a "one language to rule them all" type of guy. I feel comfortable working on the details under a shiny interface all the way up to the API, but I prefer to leave UI/UX work alone. My approach to testing is lean, meaning I usually write a test or two first, and only once a bug report is in place do I expand the test suite to cover more scenarios. Lately I've found an interest in keeping the release cycles as short as possible and thus often tinker with such tools.

Last fall I attended Hacker School in New York. An absolutely amazing learning experience that everyone who has a passion for programming should take the opportunity to apply for. Previous to that I was team lead and software developer at Malmo University making an app platform for buildings with the goal of saving energy.

In 2012, I received a M.Sc. double degree in computer science with a specialisation in distributed computing from KTH (Stockholm) and IST (Lisbon). As my final project I wrote a thesis and implemented a prototype for scaling recommendation engines at Tuenti.

I'm an active Scout since many years, most recently I led the Info/PR team for Lägr1. Beyond scouting and programming I'm a presentation nerd, food lover, active reader, avid traveller, and a big fan of smart ideas.
